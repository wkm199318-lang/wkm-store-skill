# WKM Brick Store Skill

An AI Skill that introduces **WKM Brick Store** — a LEGO marketplace store on both BrickLink and BrickOwl.

After installing this Skill, your AI assistant can answer questions about the store: what they sell, store ratings, shipping policies, VAT info, where to find them, and more.
